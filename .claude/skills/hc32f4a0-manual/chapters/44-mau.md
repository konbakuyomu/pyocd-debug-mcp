# MAU — 数学运算单元

> ✅ 本知识卡已填充。最后更新：2026-03-26

## 模块概述

MAU（Math Accelerator Unit）是硬件加速运算模块，包含开方运算和正弦运算两种运算类型。开方支持 32 位定点数输入、17 位结果输出，运算周期 8~16 个时钟；正弦支持 12 位角度输入、16 位有符号 Q15 结果输出，仅需 1 个周期。

## 关键特性

- **开方运算**：32 位输入 → 17 位输出（含四舍五入），8~16 个运算时钟周期
- **正弦运算**：12 位角度输入（360°/4096 精度）→ 16 位有符号 Q15 输出，1 个周期完成
- **定点数支持**：SHIFT[4:0] 控制开方结果左移 0~16 位，适配不同 Q 格式
- **中断/事件**：开方运算完成产生 MAU_SQRT 中断和事件输出
- **正弦无需等待**：写入角度后下一个周期即可读取结果

## 寄存器速查

> 基址: 0x40055000

| 寄存器 | 偏移 | 用途 | 关键位字段 |
|--------|------|------|-----------:|
| MAU_CSR | 0x00 | 控制状态 | START, INTEN, BUSY, SHIFT[4:0] |
| MAU_DTR0 | 0x04 | 开方输入 | SQRT_DIN[31:0] |
| MAU_RTR0 | 0x0C | 开方输出 | SQRT_DOUT[16:0] |
| MAU_DTR1 | 0x10 | 正弦输入 | SIN_DIN[11:0]（角度） |
| MAU_RTR1 | 0x14 | 正弦输出 | SIN_DOUT[15:0]（Q15 有符号） |

## 典型初始化流程

```c
/* 以开方运算为例 */

/* 1. 使能时钟 */
FCG_Fcg0PeriphClockCmd(FCG0_PERIPH_MAU, ENABLE);

/* 2. 开方运算 */
MAU_SqrtInit(CM_MAU, MAU_SQRT_SHIFT_0);  /* 不移位 */
uint32_t u32Input = 100U;
MAU_SqrtStart(CM_MAU, u32Input);
while (MAU_SqrtGetStatus(CM_MAU) == SET) {}  /* 等待 BUSY 清零 */
uint32_t u32Result = MAU_SqrtRead(CM_MAU);   /* 结果 = 10 */

/* 3. 正弦运算（无需等待） */
MAU_SinInput(CM_MAU, 1024U);  /* 90° = 4096/4 = 1024 */
uint16_t u16Sin = MAU_SinRead(CM_MAU);  /* ≈ 0x7FFF (Q15: +1.0) */
```

## 常见陷阱与注意事项

1. **开方 START 是写触发**：写 1 产生一次 START 请求，读时始终为 0
2. **BUSY 周期不固定**：根据输入数据大小，开方需 8~16 个时钟周期
3. **正弦仅需 1 周期**：写 DTR1 后等 1 个周期即可读 RTR1，无 BUSY 状态
4. **SHIFT 仅影响开方**：SHIFT 设定不影响正弦运算
5. **角度映射**：0=0°, 1024=90°, 2048=180°, 3072=270°, 4095≈360°
6. **四舍五入**：√12→3，√13→4，最低位自动四舍五入
