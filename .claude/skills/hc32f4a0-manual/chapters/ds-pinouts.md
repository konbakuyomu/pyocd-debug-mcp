## 模块概述

数据手册 Section 2 定义了 HC32F4A0 各封装的引脚配置、64 种复用功能映射、5V 耐压特性和引脚使用规范。本卡提炼硬件设计中最常查询的引脚信息，完整引脚复用表请查阅原始数据手册。

## 5V 耐压引脚

**不支持 5V 耐压的引脚（输入电压 ≤ VCC+0.3V）**：

| 引脚 | 原因 |
|------|------|
| PA0, PA1, PA2, PA6 | 模拟功能（PGA/CMP 输入） |
| PA11 (USBFS_DM), PA12 (USBFS_DP) | USB FS PHY |
| PB14 (USBHS_DM), PB15 (USBHS_DP) | USB HS PHY |

**其余所有 GPIO 均支持 5V 耐压**（VCC ≥ 2V 时 Vin ≤ 5.5V），但当 Vin > VCC+0.3V 时必须禁止内部上拉。

## Func Group（串行通信复用分组）

引脚复用 Func32~63 分为 3 组 Function Group，每个引脚只属于其中一组：

| 组 | USART | SPI | I2C | I2S | CAN |
|----|-------|-----|-----|-----|-----|
| **FG1** | 1/2/3/4 | 1/2/3 | 1/3 | 1/2 | 1/2 |
| **FG2** | 4/5/6/7 | 4/5/6 | 2/4/5 | 3 | 1/2 |
| **FG3** | 3/8/9/10 | 1/4 | 1/2/6 | 1/4 | 1/2 |

> 每个引脚在引脚功能表最后一列标注其所属 FG 编号。选择外设引脚映射时需确认该引脚的 FG 包含目标外设。

## 关键引脚使用说明

| 引脚 | 要求 |
|------|------|
| VCC | 接 1.8~3.6V，每对 VCC/VSS 就近接去耦电容；其中一个 VCC 必须接 4.7μF |
| AVCC/AVSS | 不用模拟模块时与 VCC/VSS 短接 |
| VREFH/VREFL | 不用 ADC 时 VREFH 与 AVCC 短接，VREFL 与 AVSS 短接 |
| VBAT | 不用备份电池时与 VCC 短接，外接 100nF 去耦电容 |
| VCAP_1/VCAP_2 | 就近接 VSS，电容 0.047~0.22μF（需匹配 PWC_PWRC3.PDTS 位） |
| PI13/MD | 复位解除时必须为低电平，推荐 4.7kΩ 下拉到 VSS |
| NRST | 低有效复位，不用时接电阻上拉到 VCC |
| 未用 GPIO | 悬空，或接电阻上拉/下拉 |

## VCAP 电容选型规则

| 配置 | VCAP 总容量 | PWC_PWRC3.PDTS | 特点 |
|------|------------|----------------|------|
| 两个 VCAP 引脚 | 0.094μF (2×0.047μF) | 置 1 | 快速唤醒 |
| 两个 VCAP 引脚 | 0.2μF (2×0.1μF) | 清 0 | 更好 EMC |
| 一个 VCAP 引脚 | 0.1μF | 置 1 | 快速唤醒 |
| 一个 VCAP 引脚 | 0.22μF | 清 0 | 更好 EMC |

> ESR 要求 < 0.3Ω

## 常见陷阱

1. ⚠️ 用作模拟输入时，电压不得超过 VREFH/AVCC，不受 5V 耐压保护
2. ⚠️ 5V 耐压引脚输入电压 > VCC+0.3V 时，必须禁止内部上拉/下拉
3. ⚠️ 非 5V 耐压引脚（PA0/1/2/6/11/12/PB14/15）禁止直连非本芯片电源/地，建议通过 ≥1kΩ 电阻
4. ⚠️ 芯片电源稳定后，才可将 5V 信号加到耐压引脚
5. ⚠️ VCAP 总容量必须与 PDTS 位匹配，否则掉电模式唤醒异常
6. ⚠️ 每个引脚最多 2 个功能同时有效，但不支持 2 个输出功能同时有效
7. ⚠️ 144pin 封装仅保留 PH0/PH1 和 PI12/PI13；100pin 仅保留 PH0/PH1 和 PI13
